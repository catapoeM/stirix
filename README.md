**Stirix**
 -TechStack: JavaScript(ES6+), Next.js, TypeScript, Node.js, Express.js, 
MongoDB

 • Developing a full-stack sports news platform using Node.js, Express.js, 
MongoDB, and Next.js with a focus on scalability and performance
 • Designing and implementing RESTful APIs with a modular architecture 
(controllers, routes, middlewares) following best practices
 • Implementing JWT-based authentication and role-based authorization 
(user, admin, owner) for secure access control
 • Building a content moderation system where user-generated articles are 
reviewed and approved by administrators before publication
 • Applying input validation and security mechanisms using 
express-validator and rate limiting to prevent spam and ensure data 
integrity
 • Developing a SEO-friendly frontend with Next.js (SSR/SSG) and planning 
integration with Material UI for responsive, mobile-first design
